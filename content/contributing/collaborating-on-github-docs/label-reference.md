A címkék (label) segítségével megkereshet egy adott típusú problémát. Például kattintson a táblázat címkéire, hogy megtekinthesse a github/docs repository nyitott issue-nak (problémáinak) szűrt listáját.

| Label | Leírás |
| --- | --- |
| [`help wanted`](https://github.com/github/docs/issues?q=is%3Aopen+is%3Aissue+label%3A%22help+wanted%22) | Problémák (Problems) vagy frissítések, amelyeken a közösség (community) bármely tagja elkezdhet dolgozni. |
| [`good first issue`](https://github.com/github/docs/issues?q=is%3Aopen+is%3Aissue+label%3A%22good+first+issue%22) | Problémák vagy frissítések, amelyekről úgy gondoljuk, hogy ideálisak kezdőknek. |
| [`content`](https://github.com/github/docs/issues?q=is%3Aopen+is%3Aissue+label%3Acontent) | Problémák vagy frissítések a docs.github.com webhely tartalmában. Ezek általában bizonyos Markdown ismereteket (knowledge) igényelnek. | <!-- markdownlint-disable-line search-replace -->
| [`engineering`](https://github.com/github/docs/issues?q=is%3Aopen+is%3Aissue+label%3Aengineering) | A docs.github.com webhely működésével kapcsolatos problémák vagy frissítések. Ezeknek a problémáknak (issue) a kijavításához általában némi JavaScript/Node.js vagy YAML ismerete szükséges. |<!-- markdownlint-disable-line search-replace -->
| [`codespaces`](https://github.com/github/docs/labels/codespaces)<br> [`desktop`](https://github.com/github/docs/labels/desktop)<br> [`graphql`](https://github.com/github/docs/labels/graphql) | Címkék a problémák termék vagy dokumentációs terület szerinti szűrésére. (Labels for filtering issues by a product or documentation area.) |
